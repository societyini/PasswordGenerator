# PasswordGenerator

PasswordGenerator es una herramienta escrita en Python que permite generar contraseñas aleatorias, seguras y totalmente configurables, de forma rápida y estética desde la consola.

El proyecto utiliza un archivo de configuración `society.ini` y cuenta con validaciones completas para ofrecer una experiencia de usuario limpia, clara y profesional.

## Características

* Generación de contraseñas aleatorias
* Longitud de contraseña configurable
* Soporte para mayúsculas, minúsculas, números y símbolos
* Interfaz estética en consola (colores + banner)
* Configuración mediante `password.ini`
* Validación de entrada (números, límites, errores)
* Límite máximo de generación (1–100)
* Opción de generar múltiples veces sin cerrar el programa
* Guardado automático de contraseñas en archivo `.txt`
